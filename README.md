# metabolic-disorder-prediction
Machine learning classification for metabolic disorder prediction achieving 83.8% accuracy with Random Forest. Includes unsupervised analysis (PCA, K-Means), feature importance, and model evaluation on clinical patient data.

## Academic Integrity Notice
This project was completed as part of my MSc Bioinformatics coursework. 
The dataset is not included to maintain academic integrity. 
The code demonstrates data analysis methodology and machine learning technique

## Methods
- **Unsupervised Learning**: PCA (dimensionality reduction), K-Means clustering
- **Supervised Learning**: Decision Tree, Random Forest
- **Evaluation**: 5-fold cross-validation, confusion matrices, ROC-AUC

## Results
- **Random Forest**: 83.8% accuracy, 0.81 ROC-AUC
- **Top Features**: Age, BMI, General Health

## Technologies
Python • Scikit-learn • Pandas • Matplotlib • Seaborn
